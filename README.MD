# HTML2PDF

### 运行
```
git clone https://github.com/itlad/html2pdf.git
cd html2pdf/
npm install 
npm start
```

### 使用
运行成功后哦，打开浏览器请求:  
`http://localhost:3000/pdf?sourceUrl=http://www.itlad.cn&fileName=test.pdf`
说明：  
**sourceUrl ：** 要转换为pdf文件的网页URL，必填  
**fileName  ：** 最终生成pdf文件的名字，非必填，如果不指定则以uuid为文件名
